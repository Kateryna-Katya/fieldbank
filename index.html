About the Game

Think Tetris,
but instead of clearing lines,
you're fixing up rooms

Merge Home Makeover & Design allows players to progress through merge puzzles while restoring and decorating different rooms.
Every completed task unlocks new design possibilities.
The best part? You actually see the room transform as you solve puzzles.
Didn't think I'd care about that, but it's surprisingly addictive.

How To Play

Merge, renovate, reveal

Combine Objects

Merge identical items to create upgraded tools.

Complete Design Tasks

Use resources to unlock renovation steps.

Choose Style Elements

Select between design options for each room.

Transform Spaces

Watch interiors evolve with each decision.

Unlock New Projects

Move on to larger and more creative renovations.
